# 补充说明

&emsp;&emsp;mr-Diff主要是通过依次从时间序列中提取从细到粗的趋势来进行多阶段的扩散。简单来说，其去噪过程是以一种从易到难的非自回归方式来进行的。首先，生成最粗粒度的时间趋势，然后使用扩散模型预测得到较粗粒度的时间趋势并作为细粒度趋势的推理条件，从而逐步添加更精细的条件。
